## Installation
1. Clone or download de repository:
    ```
    $ git init
    $ git clone https://github.com/oardilac/Hangman_game.git
    ```

2. Open the console inside the project directory and create a virtual environment.
    ```bash
    $ py -m venv venv
    $ source venv/Scripts/activate
    ```

3. Run the program
    ```
    $ py hangman.py
    ```
